Android Push Contacts
About the project:
  • Allows users to save contacts to phone, send SMS from the web browser/chrome extension
  • Makes use of Android C2DM to push messages to Android devices
  • Inspired by ChromeToPhone project
  • Fully open sourced, you can clone the source code from GitHub
How to use:
  • Download Android application Push Contacts (Android 2.2 is required)
  • Register the device with downloaded application
  • Use the panel to your right to send SMS
  • Add pushcontacts@appspot.com to your GTalk to receive incoming SMS
  • Chat with pushcontacts@appspot.com to reply to the latest SMS received
  • You can additionally install Chrome Extension
