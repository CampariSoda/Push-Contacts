Android Push Contacts
About the project:
  • Allows users to save contacts to phone, send SMS from the web browser/chrome extension
  • Makes use of Android C2DM to push messages to Android devices
  • Android 2.2 is required to run this application
  • Inspired by ChromeToPhone project
  • Fully open sourced, you can clone the source code from GitHub
How to use:
  • Watch this demo video
  • Download Android application with QRCode or from Android Web Market
  • Register the device with downloaded application. Once registered, you will get an invitation on GTalk from pushcontacts@appspot.com
  • Accept the invitation to receive incoming SMS
  • Chat with pushcontacts@appspot.com to reply to the latest SMS received
  • Send SMS to custom phone numbers with the following format: /sms 96969696:this is my sms
  • Use the panel to your right to send SMS
  • You can additionally install Chrome Extension
  • Follow @pushcontacts on Twitter
